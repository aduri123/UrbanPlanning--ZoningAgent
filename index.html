Output from Zoning Validation Agent

The Zoning Validation Agent produces a structured, explainable result after evaluating a parcel against zoning and planning rules.

  • Decision ‚Äî PASS / FAIL (compliance result)
  • Reasons ‚Äî Which rules were satisfied or violated (setbacks, height, FAR)
  • Measurements ‚Äî Numeric values used (setbacks, built-up %, FAR)
  • Evidence ‚Äî GeoJSON overlays, annotated map/screenshot, rule references
  • Audit & Provenance ‚Äî Data sources, timestamps, rule/graph node IDs
  • Suggested Actions ‚Äî Notify owner, request correction, schedule inspection
  • Confidence ‚Äî Confidence score for the decision
  • Machine-Readable Output ‚Äî JSON payload for downstream systems
